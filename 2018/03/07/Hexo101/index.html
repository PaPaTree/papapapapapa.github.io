Hexo101

Introduction to build static sites with Hexo for non-coders.

This tutorial is meant for macOS.

The reason why I am writing this walk-through for Hexo is that, I want to help more people like me who are interested in building a personal blog, yet having problem to find the best all-in-one tutorial online. Also, I wish to keep track of what I have learnt. If you follow as this blog instructed, you’ll be able to set up your blog and connect it with GitHub Page within few hours.

Once you have them, just open terminal and
